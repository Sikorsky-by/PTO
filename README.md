# PTO

PTO надстройка для Excel версий от 2007 до 2019 / Office 365.
В состав надстройки входят функции расчета температуры насыщения воды, энтальпии воды/пара, а так же
мелкие функции упрощающие работу.

# Установка надстройки

1.	Распаковать файлы надстройки в папку: «C:\Users\<ИмяПользователя>\AppData\Roaming\Microsoft\AddIns»
2.	Открыть Microsoft Excel.
3.	Выбрать меню Файл -> Параметры.
4.	Перейти в управление надстройками Microsoft Office.
5.	В выпадающем списке выбрать «Надстройки Excel» и нажать кнопку «Перейти…»
6.	В появившемся окне нажать кнопку «Обзор…» и в диалоговом окне выбора файла – выбрать надстройку.
7.	В окне управления надстройками Excel должен появиться пункт «ПТО энергоисточники».
8.	Выбрать из списка доступных надстроек «ПТО энергоисточники» и нажать кнопку «ОК».
9.	Надстройка установлена.
